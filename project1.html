SBAR WINDS Smart Irrigation Control System

Project Summary

Designed and built a working prototype for the SBAR WINDS Smart Irrigation Control System — a field-deployable solution for automating irrigation in guayule research plots. This effort was part of the $15M+ USDA-funded Sustainable Bioeconomy for Arid Regions (SBAR) project.

The system integrates solenoid valve control, pressure and flow measurement, and wireless data communication — coordinated by a Raspberry Pi. I defined system-level requirements with USDA and university stakeholders, sourced and integrated hardware, wrote control and data logging scripts, and produced full electrical documentation.

System Architecture

  • Primary controller: Raspberry Pi with display & GSM modem
  • Peripheral devices: Flow meters, pressure sensors, solenoids
  • Communication: RS-232 serial + I²C protocols
  • Custom PCBs: Relay control boards, analog signal converter
  • Software: Python and shell scripts for automation and logging
